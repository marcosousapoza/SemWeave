Overview

This is a sample HTML page annotated with SemWeave regions.

The annotations are invisible in the rendered page but provide structured access for AI agents.

Note: HTML comments are not rendered by browsers, making them ideal for SemWeave annotations.

API Reference

find_nodes(role?, name?, file?)

Search for nodes matching the given filters. Returns summaries only.

See Overview for general concepts.

read_node(handle)

Read the content of a node with annotations stripped.
